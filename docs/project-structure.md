# Project Structure

## Main folders
- assets/ - images, icons, and static files
- css/ - shared stylesheets
- js/ - shared JavaScript files and Supabase config scaffold
- pages/ - main HTML pages
- database/ - SQL scripts, ERD files, and seed data
- docs/ - project notes and documentation

## Pages
- pages/index.html - Home / Dashboard page
- pages/books.html - Books page
- pages/members.html - Members page
- pages/loans.html - Loans page
- pages/about.html - About page
